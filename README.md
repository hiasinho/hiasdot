##### Installation

From your homedirectory (on Linux/MacOSX):

* `cd ~ && mkdir -p git && cd git`
* `git clone git://github.com/hiasinho/hiasdot.git`
* `ln -sfn hiasdot ~/.dotfiles`
* `ln -sfn hiasdot/bash_profile ~/.bash_profile`
